# Usage
<!-- usage -->
```sh-session
$ npm install -g aws-pinpoint
$ aws-pinpoint COMMAND
running command...
$ aws-pinpoint (-v|--version|version)
aws-pinpoint/0.0.0 darwin-x64 node-v10.9.0
$ aws-pinpoint --help [COMMAND]
USAGE
  $ aws-pinpoint COMMAND
...
```
